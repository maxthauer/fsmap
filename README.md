# fsmap

A vibe coded Flask app using the SimConnect library to track your FS2024 gameplay on a map. The Flask app will listen on `localhost:5000`

## Install dependencies

`pip install Flask flask-socketio Python-SimConnect`

## Run it

`python app.py`

Launch map.html in your browser:
